QUICK START - BUILD THE EXE
================================================================================

METHOD 1: Using the Batch File (Easiest)
-----------------------------------------
1. Double-click: BUILD_PORTABLE.bat
2. Wait 2-5 minutes for the build to complete
3. Done! Find your exe: CameraFinder_Portable.exe

METHOD 2: Using Python Command
-------------------------------
1. Open Command Prompt in this folder
2. Run: python build_portable_exe.py
3. Wait for build to complete
4. Done! Find your exe: CameraFinder_Portable.exe

================================================================================
REQUIREMENTS BEFORE BUILDING
================================================================================

Make sure you have:
1. Python 3.7 or higher installed
2. All packages installed: pip install -r requirements.txt
3. ffprobe.exe in this folder (already present)
4. camera_gui_fixed.py (already present)

================================================================================
WHAT HAPPENS WHEN YOUR FRIEND RUNS THE EXE?
================================================================================

1. They double-click CameraFinder_Portable.exe
2. A window opens with the Camera Finder interface
3. They click "Scan for WiFi Cameras" or "Quick IP Scan"
4. A popup appears asking for:
   - Username (default: admin)
   - Password (their camera password)
5. The app scans the network for cameras
6. Working RTSP URLs are displayed
7. They can copy the URLs to use in VLC or other players

================================================================================
TESTING THE BUILD
================================================================================

After building, test on YOUR computer first:
1. Double-click CameraFinder_Portable.exe
2. Enter camera credentials when prompted
3. Click scan button
4. Verify it finds cameras and shows RTSP URLs

If it works on your computer, it will work on theirs!

NEED HELP?
================================================================================

Common issues and solutions:

Issue: "ffprobe.exe not found"
Solution: Make sure ffprobe.exe is in the same folder before building

Issue: "ONVIF WSDL files missing"
Solution: Reinstall onvif-zeep: pip install --upgrade onvif-zeep

Issue: Build succeeds but exe doesn't work
Solution: Try building with console=True in the spec file to see errors

Issue: Exe is flagged by antivirus
Solution: This is normal for PyInstaller exes. Create exception or sign the exe.
